How to Open JAR Files on Android

Use Jar File Opener to open, browse and inspect Java JAR archives on Android.

Steps

  1. Install Jar File Opener.
  2. Tap Open JAR File.
  3. Select a .jar file.
  4. Browse folders and open supported entries.

What you can view

  • META-INF/MANIFEST.MF
  • .class files
  • Text files
  • Images and resources
