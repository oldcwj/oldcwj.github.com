How to Extract JAR Files on Android

Use Jar File Opener as a JAR extractor for Android.

Steps

  1. Open a JAR file.
  2. Browse archive contents.
  3. Select files or extract all.
  4. Save extracted contents to your device.

Common files

  • .class files
  • Manifest files
  • Text, JSON and XML
  • Images and resources
