About Minecraft RCON Admin

Minecraft RCON Admin helps server owners connect to Minecraft Java Edition servers and run RCON commands from Android.

Key Features

  • Connect to Minecraft RCON servers
  • Run remote admin commands
  • Manage server sessions
  • Save server profiles
  • Use Android as a server console
  • Simple tool for Minecraft admins
