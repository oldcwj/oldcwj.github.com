How to Open OBB Files on Android

Use OBB File Opener to view file details, browse contents and extract supported files.

Steps

  1. Install OBB File Opener.
  2. Tap Open OBB File.
  3. Select a .obb file.
  4. Browse internal contents or extract files.

Important

Only extract files you own or have permission to access. Respect app copyrights, game licenses and developer terms.
