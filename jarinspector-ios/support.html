JarInspector Support

Help and contact information for JarInspector on iPhone.

About JarInspector

JarInspector is an iPhone app for opening JAR, ZIP, and Java class files. It can inspect archive contents, show manifest information, browse classes, and preview decompiled Java source locally on your device.

Contact Support

If you need help, want to report a bug, or have a feature request, contact us by email.

Email: imzine.com@gmail.com

When reporting an issue, please include the app version, your iPhone model, iOS version, the file type you were opening, a short description of the issue, and steps to reproduce it.

Before Sending Files

JarInspector processes files locally. Please do not send private, confidential, or sensitive JAR, ZIP, or class files unless they are necessary for troubleshooting and you are comfortable sharing them.
