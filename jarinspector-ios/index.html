JarInspector iOS

Open, inspect, and decompile JAR, ZIP, and Java class files locally on iPhone.

About JarInspector

JarInspector is an iPhone utility for developers, students, and file analysis workflows. It helps you open Java archive files, browse archive contents, view manifest metadata, inspect class information, and preview decompiled Java source.
