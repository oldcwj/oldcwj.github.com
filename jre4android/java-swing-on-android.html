Run Java Swing Apps on Android

Launch desktop-style Java GUI apps and interact with them using touch controls.

How it works

Open a compatible Swing JAR with Jre4Android and use virtual mouse, zoom, move and keyboard controls to interact with the GUI.

Compatibility

Some Java desktop apps may require external libraries or desktop-specific APIs. Test each app on your device.
