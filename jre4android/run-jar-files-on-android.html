How to Run JAR Files on Android

Use Jre4Android to launch compatible Java JAR apps directly on your Android device.

Steps to run a JAR file

  1. Install Jre4Android from Google Play.
  2. Open the app.
  3. Tap Open JAR File.
  4. Select your .jar file.
  5. Tap Run and view terminal or GUI output.

Common problems

If the JAR does not start, check whether it includes Main-Class in META-INF/MANIFEST.MF. You can use Jar File Opener to inspect the manifest.
