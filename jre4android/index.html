About Jre4Android

A Java runtime tool for Android that helps you run Java JAR files, CLASS files, command-line Java programs, Swing GUI apps and classic J2ME apps.

Key Features

  • Run JAR files on Android
  • Run .class files
  • Java command-line mode
  • Java Swing GUI support
  • J2ME app support
  • Java runtime on Android
